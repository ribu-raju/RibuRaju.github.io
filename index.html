Projects

Featured work.

Final Year Project · IoT & Embedded Systems
Real Time Feedback from Smart Devices Using an IoT Platform & TTP223 Touch Sensor
🎓 APJ Abdul Kalam Technological University 📅 June 2020

An IoT-based home automation system enabling real-time control of home appliances via a Blynk Android application and TTP223 capacitive touch sensors — with bidirectional feedback between physical switches and the app.

NodeMCU (ESP8266) TTP223 Touch Sensor 4-Channel Relay Blynk App Arduino IDE Wi-Fi / IoT EasyEDA (PCB) C / C++
  • Designed and built a custom PCB using EasyEDA with double-sided copper cladding to minimize board size and component complexity.
  • Integrated TTP223 capacitive touch sensors as physical switches that sync in real time with the Blynk Android app — touching a sensor immediately reflects the updated state in the app.
  • NodeMCU (ESP8266) microcontroller manages Wi-Fi connectivity, processes touch inputs, and drives relay modules to control AC appliances.
  • Implemented OTA (Over-the-Air) firmware updates so the device can be reprogrammed remotely without physical access.
  • Designed the entire home automation circuit schematic and PCB layout independently using EasyEDA software.
